## Neovim Notes

Neovim is sourced from this directory. Stored in this repository, but not in the nix store. This makes it easier to make quick changes without rebuilding the environment.

### Sidekick & Copilot
- :sidekick nes disable - Disables normal mode Copilot nes suggestions
- :Copilot disable - Disables insert mode Copilot completion

### LSP
- [d next diagonstic/error
- ]d previous diagnostic/error
- K show documentation
- <Leader>ca Show code actions
